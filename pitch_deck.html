AnfieldVoice • Security Architecture

Layered security for every entry point

Every visitor, delivery, and access event is verified, logged, and auditable — from the gate to the apartment door.

Gate

PIN-Based Visitor Ticketing

Residents generate unique one-time or recurring PINs per visitor. Entry requires valid PIN + security verification at the terminal. No PIN = no entry.

Recurring

Recurring Visitor Profiles

Domestic workers, gardeners, school transport — saved profiles with vehicle details, photos, and schedules. Security admits by recognition in seconds.

Audit

Complete Audit Trail

Every admission, denial, delivery drop-off, and PIN generation is timestamped and logged. Searchable by resident, visitor, vehicle, or apartment.

Encryption

End-to-End Protected

SIP signalling over TLS 1.3. Media streams encrypted with SRTP (AES-256). All API traffic over HTTPS. Database connections with certificate pinning.

Identity

Role-Based Access Control

Residents, Property Administrators, Security, Body Corporate — each role has precisely scoped permissions. Additive roles, no overlap, full accountability.

Privacy

POPIA Compliant by Design

Optional resident directory (opt-in only). Visitor data retained only as long as necessary. Complete data export and deletion on request.

AnfieldVoice • Delivery Management

Delivery without the chaos

From groceries to couriers to large-item drop-offs — one system to manage every package that enters the estate.

📦

Real-Time Delivery Notifications

Driver arrives at gate → scans or enters apartment → resident gets instant push notification with courier, vehicle, and ETA. No more "did my parcel arrive?" calls to security.

🔄

Approve, Redirect, or Deny

Resident decides from their phone: allow delivery to the door, redirect to security office for collection, or deny entry. All recorded in the audit trail.

📋

Delivery History & Proof

Complete searchable log of every delivery event — what arrived, when, from whom, and whether it was collected or delivered. Resident sees their own history; body corp sees estate-wide.

Security Integration

Security Dashboard

Gate officer sees all active deliveries, expected arrivals, and recurring visitors on one real-time screen. Admit, hold, or escalate with one click. Shift handover logs preserve continuity.

Future

Smart Locker Integration

Contactless 24/7 package retrieval via integration with delivery lockers. Driver drops, resident collects with PIN or QR code — no security involvement needed.
